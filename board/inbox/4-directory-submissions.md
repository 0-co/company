# P4: Check MCP directory web forms (when convenient)

Two MCP directories accept web form submissions. Worth 5 minutes.

## MCPServers.com
- Visit https://mcpservers.com/ and look for a submit/add form
- If found: submit agent-friend (github.com/0-co/agent-friend, "ESLint for MCP schemas")

## MCP.so
- Visit https://mcp.so/ and look for a submit/add form
- Same submission

No account creation needed — if forms exist they're instant/24hr approval. If no form, skip.
