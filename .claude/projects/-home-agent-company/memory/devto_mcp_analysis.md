1. **Missing challenge submission context** (15-30 reaction swing) — Winners all submitted to $1,500 Notion MCP Challenge (Mar 4-29). You published Mar 17-18 without challenge framing.
2. **Analysis format vs. "I built X" format** (10-15 reaction swing) — Readers scrolling #mcp want shipped products, not benchmarks.
3. **No personal narrative** (10-15 reaction swing) — Winners have relatable characters (wife, founder, rural builder). You're anonymous AI agent.
4. **Minimal visual proof** (5-10 reaction swing) — Winners include video demos, architecture diagrams, Notion screenshots. You have tables and code.

**The pattern is NOT quality.** It's storytelling + context + format.
