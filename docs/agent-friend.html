agent-friend

A composable personal AI agent library. Not a platform you run — a library you import.
zero required deps Anthropic + OpenAI 136 tests MIT license

The pitch

Every "personal AI agent" project is a platform you install and run. OpenClaw. PocketPaw. Agent Zero. They're opinionated runtimes with their own UX.

If you want to build your own agent — with your own tools, your own memory strategy, your own model routing — you're back to square one, wiring up SQLite and subprocess from scratch.

agent-friend is the library for that. The primitives, not the platform.

Quick start

# Free tier — no credit card required (OpenRouter)
pip install "git+https://github.com/0-co/agent-friend.git[all]"
export OPENROUTER_API_KEY=sk-or-...  # free at openrouter.ai

# Or with Anthropic / OpenAI
export ANTHROPIC_API_KEY=sk-ant-...

# Interactive REPL — watch tools execute in real time
agent-friend -i --tools search,memory,code

# Or one-shot
agent-friend "search for the latest AI agent news"
# Or use it in code
from agent_friend import Friend

friend = Friend(
    seed="You are a helpful personal AI assistant.",
    tools=["search", "code", "memory"],
    model="claude-haiku-4-5-20251001",
    budget_usd=1.0,
)

response = friend.chat("Search for recent Python agent frameworks and summarize")
print(response.text)

# Multi-turn — history is preserved automatically
friend.chat("My name is Alice")
friend.chat("What is my name?")  # Returns: Alice

# Or load from a YAML config
friend = Friend.from_yaml("friend.yaml")

Tools

MemoryTool

SQLite + FTS5 persistent memory. remember(), recall(), forget(). Survives restarts.

CodeTool

Sandboxed Python + bash via subprocess. Captures stdout/stderr. Configurable timeout.

SearchTool

DuckDuckGo HTML scraper. No API key. Returns titles, URLs, snippets.

BrowserTool

agent-browser delegate. Returns page text. Graceful fallback if not installed.

EmailTool v0.2

AgentMail integration. Read inbox, send email (draft by default). Free tier: 3 inboxes.

# Use tool instances for custom config
from agent_friend import Friend, MemoryTool, CodeTool, SearchTool

friend = Friend(
    tools=[
        MemoryTool(db_path="~/.my_agent/memory.db"),
        CodeTool(timeout_seconds=10),
        SearchTool(max_results=5),
    ]
)

Config file

# friend.yaml
seed: "You are a helpful personal AI assistant."
model: claude-haiku-4-5-20251001
tools:
  - search
  - code
  - memory
memory_path: ~/.agent_friend/memory.db
budget_usd: 5.00
max_context_messages: 20
friend = Friend.from_yaml("friend.yaml")

ChatResponse

response = friend.chat("Search for something")

response.text          # The final answer
response.tool_calls    # What tools were called
response.input_tokens  # Tokens used
response.output_tokens
response.cost_usd      # Estimated cost
response.model         # Model that responded
