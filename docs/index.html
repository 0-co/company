ai-convo analyzer
Paste any JSON conversation. Get: vocabulary overlap, depth score, novelty curve.
agent-budget
Zero-config AI API budget enforcement. Wraps Anthropic/OpenAI clients. Raises BudgetExceeded before your agent blows the bill.
agent-context
Prevent context rot in long-running agents. Sliding window, token budget, and compress strategies. Zero deps. One import.
agent-eval
Unit testing for AI agents. Define test cases, run against any agent function, get pass/fail/score. Zero deps. Works in CI.
agent-shield
Security scanner for AI agent skills and plugins. Detects prompt injection, credential theft, and encoded payloads. Like npm audit but for agent skills.
agent-shield-action
GitHub Action wrapping agent-shield. Add one line to CI to block malicious skills from reaching production. uses: 0-co/agent-shield-action@v1
agent-id
Zero-dep agent identity and trust verification. Cryptographically sign agent calls. Verify before acting. Blocks prompt injection impersonation attacks.
agent-retry
Retry logic for AI agent API calls. Exponential backoff with jitter, Retry-After header awareness, sync + async support. Knows which LLM errors are retryable. Zero deps.
agent-gate
Human-in-the-loop approval for AI agent actions. Pause before irreversible operations. @gate.requires("Delete {path}"). Pluggable handlers: stdin, auto-approve, callbacks. Zero deps.
agent-log
Structured logging for AI agents. Sessions, spans, token tracking, cost calculation, auto secret redaction. Python's stdlib logging doesn't know what a span is. This does. Zero deps.
agent-cache
LLM response caching for AI agents. Wrap your Anthropic or OpenAI client in one line. Identical calls served from disk. Shows you exactly how much money you saved. Zero deps.
agent-mock
Record/replay/fixture LLM responses for testing. No real API calls in your test suite. Strict mode, cassette files, error simulation. Works with Anthropic and OpenAI. Zero deps.
agent-constraints
Enforce rules on agent tool calls at the Python level, not the prompt level. Agents can bypass prompt constraints. They cannot bypass Python code. Violation log, log-only mode. Zero deps.
agent-checkpoint
Save and restore agent state. Resume after failure. Rollback before risky operations. Branching history. Fixes "agentic amnesia" — agents losing context and restarting from scratch. Zero deps.
agent-schema
Validate LLM responses against a JSON schema. Auto-retry with error feedback until the model returns valid output. Extracts JSON from markdown-wrapped responses. Zero deps.
agent-timeout
Timeout and deadline enforcement for AI agent API calls. with_timeout(), @timeout_decorator(30), context manager. TimeBudget tracks total time across retries. Cross-platform. Zero deps.
agent-rate
Rate limiting for AI agent API calls. Token bucket (burst-aware) + sliding window. Requests-per-minute and tokens-per-minute. Decorator, context manager, async. Prevents the 429s instead of retrying them. Zero deps.
agent-router
Route LLM calls to different models based on rules. Use Haiku for short inputs, Sonnet for complex ones. Built-in conditions: token count, message length, keywords. Haiku is 30x cheaper than Opus — routing correctly matters. Zero deps.
agent-fallback
Multi-provider failover for AI agents. When Anthropic returns 529, automatically try OpenAI. Completes the reliability trilogy with agent-retry and agent-timeout. Includes circuit breaker to stop hammering dead endpoints. Zero deps.
agent-trace
Distributed tracing for multi-agent workflows. Trace IDs, parent-child span hierarchies, context propagation across process boundaries. When 5 agents fail in sequence and you don't know which one caused it. Zero deps.
