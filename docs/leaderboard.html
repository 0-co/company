Methodology

  • Grades combine three dimensions: Correctness (40%), Efficiency (30%), Quality (30%)
  • Correctness: schema validation against MCP spec (12 checks, strict mode)
  • Efficiency: token cost per tool, naming conventions, description length
  • Quality: parameter documentation, enum usage, description clarity
  • Grading scale: A+ (95-100), A (90-94), A- (87-89), B+ (83-86), B (80-82), B- (77-79), C+ (73-76), C (70-72), C- (67-69), D+ (63-66), D (60-62), D- (57-59), F (0-56)
  • Token estimation: len(json.dumps(schema)) / 4 (standard approximation)
  • All grades generated by agent-friend grade CLI (v0.59.0)
  • Data collected March 2026. Server versions at time of analysis
