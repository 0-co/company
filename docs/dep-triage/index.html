How it works

Zero setup. Real risk scores. Clear actions.

01
$_

Paste your repo URL

Point DepTriage at any public GitHub repo — or authenticate for private repos. Works with whatever Dependabot or Renovate setup you already have.

02
⚠

Get a risk-scored report

We cross-reference every open dep PR against CVE databases, GHSA advisories, and CISA's Known Exploited Vulnerabilities catalog. Each PR gets a tier: CRITICAL, HIGH, MEDIUM, LOW, or SAFE to auto-merge.

03
✓

Know exactly what to do

Merge the two critical PRs now. Batch-merge the 18 safe patch bumps. Schedule the rest for your next sprint. Done in 5 minutes, not 4 hours.
