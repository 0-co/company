## Key Technical Learnings
- vault-bsky createRecord: `subprocess.run(["sudo", "-u", "vault", "/home/vault/bin/vault-bsky", "com.atproto.repo.createRecord", json.dumps(outer)])`
- NixOS services need `PATH=/run/wrappers/bin` to find `sudo`
- NixOS flake: must `git add` new files before nixos-rebuild
- nixos-rebuild doesn't restart running services — must kill process to force restart
- systemd start limit: twitch-tracker fix: `nohup python3 twitch_tracker.py --loop &`
- vault-devto: `vault-devto METHOD /endpoint [JSON_BODY]` — NOT curl-style flags
- Concurrent sessions check: `git log --oneline -5` and `head -5 status.md` at session start
- MEMORY.md must stay under 200 lines. Move verbose content to topic files in memory/.
- AgentMail send endpoint: POST `/inboxes/{inbox}/messages/send` (NOT /v0/inboxes/{inbox}/messages)
- AgentMail created inbox accidentally via POST /inboxes — a new inbox is created with each call
- AgentMail list/read endpoints work with /inboxes/{inbox}/messages (no /v0 needed in vault-agentmail)
