# Alternative Distribution Channels for MCP Tool Linter/Optimizer
_Research date: 2026-03-17_

---

## Summary

- **Biggest finding**: There are 10+ MCP-specific directories/registries that accept free submissions -- this is the highest-leverage channel since they aggregate exactly the audience who would use a tool linter
- **Best immediate action**: Submit to mcp.so, PulseMCP, mcpservers.org, MCP Market, and mcpserverfinder.com (all free, all accept web form submissions, no human verification needed)
- **Overlooked channel**: DevHunt.org -- a Product Hunt alternative specifically for developer tools, uses GitHub PR-based submissions (no account gatekeeping), and the MCP/AI agent community is its core audience

## Key Insight

The MCP ecosystem has exploded to 10,000+ servers but the discovery/quality infrastructure is fragmented across 10+ directories. Each directory is hungry for listings. This is a rare window where submitting to every directory is high-ROI because:

1. Each submission takes 5-15 minutes
2. The directories are actively curating and competing for completeness
3. agent-friend occupies a unique niche (build-time linting vs. runtime tools)
4. Zero competition in the "MCP schema linter" category across any directory

The MCP Official Discord (11,658 members) is the single most valuable community channel. Every person there is either building or consuming MCP tools -- the exact audience for a token cost auditor.
