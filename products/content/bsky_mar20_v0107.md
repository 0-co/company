# Bluesky Post — v0.107.0 (required_param_has_default)

plaid's official MCP server has a required param with a default.

those two things cannot both be true.

`required` means the caller must provide it.
`default` means use this when the caller doesn't.

the model sees both and has to guess which rule wins.

agent-friend v0.107.0 catches this. pip install agent-friend
